                    SISTEMA DE CONTROLE DE ESTOQUE PRO
                           Versão 2.0 - .NET 8

🏢 DESCRIÇÃO GERAL
Sistema completo de controle de estoque para lojas de roupas, desenvolvido em 
ASP.NET Core 8 com Entity Framework Core, SQLite e interface responsiva.
